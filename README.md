# EduTec - Backend 

# Descripción - Eductec es un sistema tipo portal educativo en desarrollo.  
Este repositorio corresponde al módulo backend, encargado de la lógica del sistema, autenticación y manejo de datos.

# Objetivo - Desarrollar un sistema backend confiable que permita:

- Autenticación de usuarios con sus respectivos permisos (Profesorado, administración y estudiantes)
- Gestión de inscripciones
- Consulta de información académica 
- Comunicación segura con el frontend

## Requerimientos - 
### Requerimientos Funcionales

- Registro de usuarios
- Inicio de sesión (Login)
- Validación de credenciales
- Gestión de inscripciones
- Almacenamiento de datos de usuarios
- Conexión con el frontend
- Sistema de inscripciones
- Consulta  de calificaciones
- Comunicación con el frontend

### Requerimientos No Funcionales

- Seguridad en el manejo de datos
- Interfaz de programación clara (API REST)
- Buen rendimiento
- Código mantenible
- Escalabilidad básica
- Disponibilidad del sistema 
- Documentación del sistema

## Tecnologías

- Lenguaje: Java
- Framework: Spring Boot
- Base de datos: MySQL
- Control de versiones: Git y GitHub
